# **1. Installation and Environment Setup for Frontend and Backend**  
*Getting your machine ready for the Apparel Classifier project*

---

### Objectives
- Install all required runtime and tools (Python, Node.js, VS Code, Git)
- Set up the Python virtual environment and install backend dependencies
- Scaffold the React frontend and install its packages
- Verify that both environments are correctly configured

---

### 1. Prerequisites
Before we start writing a single line of code, ensure your system has the following:

| Tool          | Minimum Version | Purpose                                 |
|---------------|-----------------|-----------------------------------------|
| Python        | 3.7 or higher   | Run the Flask backend and train the model |
| Node.js + npm | LTS (≥ 16.x)    | Build and run the React frontend        |
| Git           | any recent      | Version control (optional but recommended) |
| VS Code       | latest          | Code editor                             |

**Verify installations** (open a terminal):
```bash
python --version        # or python3 --version
node --version
npm --version
git --version
```
If any are missing, download them from the official websites.

### 3. Backend Setup (Python + Flask + TensorFlow)

We’ll isolate dependencies using a Python virtual environment.

#### 3.1 Create and Activate Virtual Environment
```bash
cd backend
python -m venv venv
```
- **Windows**: `venv\Scripts\activate`
- **macOS/Linux**: `source venv/bin/activate`

After activation your terminal prompt should show `(venv)`.

#### 3.2 Install Dependencies
Create a file `backend/requirements.txt` with the following content:

```
flask
flask-cors
tensorflow
pillow
numpy
```

Then run:
```bash
pip install -r requirements.txt
```

**What each package does:**
- `flask` – lightweight web framework to serve prediction API
- `flask-cors` – enables Cross-Origin Resource Sharing so the frontend can call the backend
- `tensorflow` – loads the trained model and makes predictions
- `pillow` – handles image loading and resizing
- `numpy` – numerical operations on image arrays

> ⚠️ **Note for Windows users**: TensorFlow may require Microsoft Visual C++ Redistributable. If you encounter DLL load errors, install it from [Microsoft](https://aka.ms/vs/17/release/vc_redist.x64.exe).

#### 3.3 Verify Backend Installation
Create a simple test file `backend/test_env.py`:
```python
import flask
import flask_cors
import tensorflow as tf
import PIL
import numpy as np
print("Flask version:", flask.__version__)
print("TensorFlow version:", tf.__version__)
```
Run `python test_env.py`. If no errors appear, your backend environment is ready. Delete the test file afterwards.

### 4. Frontend Setup (React + Axios)

We’ll use Create React App to bootstrap the frontend.

#### 4.1 Create React App
Navigate back to the project root, then:
```bash
npx create-react-app frontend
```
This will populate the `frontend` folder with a default React project. If you already have a `frontend` folder from the structure, remove it first or run `npx create-react-app frontend` inside the parent directory – it will replace the empty folder.

#### 4.2 Install Additional Dependency
The app uses `axios` to send HTTP requests to the backend.
```bash
cd frontend
npm install axios
```

#### 4.3 Replace the Main Component
The frontend code provided in the project brief (`App.js`) must replace the default `src/App.js`.  
Copy the React component code (shown in the original material) into `frontend/src/App.js`, overwriting its contents.

**Quick sanity check:** Start the development server to make sure the React app compiles:
```bash
npm start
```
It should open `http://localhost:3000` in your browser and display the Apparel Classifier UI (even though the backend isn't running yet, the upload and buttons will be visible).

Stop the server with `Ctrl+C` after the check.

### 5. Training Environment (Optional for Now)
Later we will train the deep learning model. The `training/` folder needs the same Python packages as the backend plus possibly a dataset. For now, you can install the same dependencies in a separate virtual environment inside `training/` (or reuse the backend’s environment if you prefer, but isolation is cleaner).

Create and activate a virtual environment in `training/` and run:
```bash
pip install tensorflow pillow numpy
```

### 6. Troubleshooting Common Installation Issues

| Problem | Quick Fix |
|---------|-----------|
| `python` not recognized on Windows | Ensure Python is added to PATH during installation, or use `py` command. |
| `pip` not recognized | Run `python -m pip install --upgrade pip` |
| TensorFlow installation fails | Upgrade pip: `pip install --upgrade pip`. On Apple Silicon (M1/M2) consider `tensorflow-macos` and `tensorflow-metal`. |
| `create-react-app` takes too long | Use `npx create-react-app frontend --use-npm` (default) or check network. |
| Port 3000 already in use | React will prompt to use another port or kill the existing process. |
